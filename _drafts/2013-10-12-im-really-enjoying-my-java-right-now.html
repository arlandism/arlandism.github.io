--- layout: post title: date: '2013-10-12T13:08:00.000-07:00' author: aml tags: modified_time: '2013-10-12T13:08:52.707-07:00' blogger_id: tag:blogger.com,1999:blog-1730093662423317857.post-3072233185080984679 blogger_orig_url: https://immutablearlandis.blogspot.com/2013/10/im-really-enjoying-my-java-right-now.html --- I'm really enjoying my Java right now. Fortunately, I'm reading "Refactoring" right now which, to me, is beginning to imply that unless a system's design is absolutely terrible it can probably be refactored. This has vast implications in my day-to-day workflow since I worry less about decisions I make in the heat of the moment.

This is a bit of a double-edged sword though, since reminding myself that I can always come back to some rotten code makes writing the code more palatable. It's important to keep this attitude in check since in the real world, those "I'll come back to it" moments probably happen pretty frequently, while the "I'm back" moments probably happen very rarely.
